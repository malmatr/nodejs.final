# NodeJS Final Project

## Description

This project is a basic Node.js and Express server with a structured backend and a simple frontend.

## Features

- Express server running on port 3000
- MVC folder structure (routes, controllers, models)
- Multiple API routes
- Frontend connected to backend
- Environment variable support

## Installation

1. Clone the repository
2. Run `npm install`
3. Create a `.env` file with:
   PORT=3000
4. Run `npm run dev`

## API Routes

- GET /api/hello → returns text response
- GET /api/status → returns JSON server status

## Frontend

Open `frontend/index.html` in your browser to interact with the backend.
